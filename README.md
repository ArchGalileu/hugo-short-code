# my custom shortcodes for hugo
* img.html  
usage;
`{{< img src="/img/image.jpg" >}}`
**responsive image with %100 width**

* embedvimeo.html  
usage;
`{{< embedvimeo 224893042 >}}`
**responsive vimeo video**

* embedyoutube.html  
usage;
`{{< embedyoutube 9AFtdrCFyd0 >}}`
**responsive youtube video**
